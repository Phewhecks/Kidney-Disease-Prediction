# Diagnosys

Diagnosys is a web application built using **HTML**, **CSS**, **JavaScript**, and **Python** with **Flask** as the backend framework. The platform is designed to assist doctors in providing faster and more accurate analysis of lab data, ultimately improving the diagnostic process.

## Features

Quickly input test results, generate automated health reports, and receive personalized lifestyle, diet, and fitness recommendations based on patient data.

## Technologies Used

* **Frontend**: HTML, CSS, JavaScript
* **Backend**: Python, Flask
* **Database**: SQLite
* **Libraries**:

  * **NumPy** and **Pandas**
  * **Scikit-learn** 
  * **Werkzeug**
  * **ReportLab**
  * **Plotly** 

## Setup Instructions

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/upadhyayabhisek/Diagnosys.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Diagnosys
   ```

3. Create a virtual environment and activate it:

   ```bash
   python3 -m venv venv
   venv\Scripts\activate
   ```

4. Install the required Python libraries:

   ```bash
   pip install -r requirements.txt
   ```

5. Set up your MySQL database and configure it in the application.

6. Run the Flask application:

   ```bash
   python app.py
   ```
